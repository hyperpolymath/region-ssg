== Why region-ssg?
=== Region-Based Partitioning
ZPL allows the definition of `regions` that govern how array operations apply. In `region-ssg`, your site is partitioned into these regions, allowing the compiler to automatically allot computations to different processors.

=== Implicit Concurrency
Because ZPL uses an array abstraction for a vector-oriented model, it achieves concurrency without parallel directives. Parallelism arises naturally from the semantics of your site transformations.
